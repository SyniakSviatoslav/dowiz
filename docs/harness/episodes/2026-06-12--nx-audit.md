## Error patterns extracted

10 patterns identified: Schema-Query Mismatch, Library API Drift, Incomplete Event Wiring, Connection Lifecycle Leak, Resilience Gap, Backward Compat Blindspot, Topology Ignorance, Permission Assumption, Missing Pre-flight Check, Code Duplication. (Full detail in `NX-RETRO.md` §2.)

## Key insight

Only 1 of 20 issues was catchable by TypeScript. 17 were catchable by tests. The dominant failure category is **runtime environmental mismatch** — wrong pool type, wrong DB port, wrong privileges, wrong library version — which neither types nor unit tests detect. Integration tests + topology probes are the only defense.
